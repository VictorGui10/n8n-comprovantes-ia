Leitor Inteligente de Comprovantes com IA

Tecnologias:

- n8n
- Google Gemini
- Telegram Bot API
- Google Sheets

Fluxo:

Telegram
↓
Download da imagem
↓
Gemini OCR + IA
↓
Extração em JSON
↓
Google Sheets

O sistema recebe comprovantes enviados pelo Telegram, extrai automaticamente:

- Data
- Estabelecimento
- Valor
- Categoria
- Descrição

e registra tudo automaticamente em uma planilha.
